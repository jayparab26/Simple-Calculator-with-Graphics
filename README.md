# 📱 Simple Calculator with Graphics

A sleek, responsive, and modern web-based calculator designed with a clean pastel aesthetic and intuitive layout groupings. Built using pure HTML5, CSS3 (featuring advanced CSS Grid layout controls), and vanilla JavaScript.

![Calculator Preview](preview.png)

## ✨ Features

- **Fluid Grid Layout:** Clean layout featuring structured spacing, comfortable button padding, and responsive scaling.
- **Sleek Display Screen:** Matte slate screen styling with custom color configurations to provide a premium, native-app depth feel.
- **Mobile First Design:** Fully responsive layout that automatically adjusts to look beautiful across desktops, tablets, and mobile screens.
- **Utility Support:** Includes seamless clear (`C`) and single-character deletion (`Undo`) functionality.

## 🛠️ Built With

* **HTML5:** Structured semantic markup utilizing containers and action buttons.
* **CSS3:** Premium 135-degree pastel gradients, smooth micro-interactions, custom fonts, and a responsive **CSS Grid** layout.
* **JavaScript (ES6):** Fast, lightweight input handlers managing real-time mathematical evaluations under the hood.
